# Defalt's Block Selector

Java mod with primary purpose of customize the block selector outline.

## Features

- Client-side only
- In-game config screen via Mod Menu + Cloth Config
- Outline color in ARGB
- Outline thickness control
- Outline glow control
- Color modes:
    - `Static`
    - `Rainbow`
    - `Gradient (2 Colors)`
    - `Pulse`
    - `Breathing`
- Animation speed control

## Requirements

- Fabric API
- Mod Menu
- Cloth Config API

## Configuration

Open `Mod Menu -> Defalt's Block Selector -> Configure`.

### Available settings

- `Color Mode`: `Static`, `Rainbow`, `Gradient (2 Colors)`, `Pulse`, `Breathing`
- `Primary Color` (ARGB)
- `Secondary Color (Gradient/Pulse)` (ARGB)
- `Animation Cycles Per Second` (for animated modes)
- `Outline Thickness`
- `Outline Glow`

Config file path:

- `config/defalt-block-selector.json`
